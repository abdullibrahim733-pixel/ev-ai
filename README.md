# EV-AI ⚡🤖

**Autonomous Electric Vehicle — 3D AI Driving Simulator**

[![Live Demo](https://img.shields.io/badge/Live-Demo-00e5ff?logo=github)](https://abdullibrahim733-pixel.github.io/ev-ai/)

An AI-controlled electric vehicle built with p5.js WEBGL 3D. The car drives autonomously using sensor raycasting, path following, and obstacle avoidance.

Loads a **Python-generated 3D model** via a FastAPI backend, with procedural fallback.

## 🎥 Demo Video

[Watch the coding demo](https://github.com/abdullibrahim733-pixel/ev-ai/releases) showing the entire project being built.

## 🧠 How It Works

### 3D Electric Vehicle
Generated by Python (`generate_model.py`), served via FastAPI backend:
- **GLTF Binary** (`.glb`, 19.5 KB) — standard 3D format
- **Wavefront OBJ** (`.obj`, 42.9 KB) — p5.js compatible
- **Procedural fallback** — works standalone without backend

### AI Agent Controller
| System | Description |
|--------|-------------|
| **Sensors** | 8 directional raycast sensors (front, sides, rear) |
| **Path Following** | Pure pursuit algorithm with 64 waypoints |
| **Obstacle Avoidance** | State machine: follow → avoid → recover |
| **Speed Control** | Adaptive to path curvature |
| **Battery Management** | Eco mode below 15%, range calculation |

### Professional UI
CAD/Simulation-grade interface (Autodesk/OnShape design language):
- Application-style top menu bar
- Vertical SVG icon toolbar
- Tabbed left panel (Dashboard / Sensors / AI Agent)
- SVG circular gauges with real-time animation
- Viewport HUD + orientation cube
- Status bar with coordinates, camera mode, FPS

## 🚀 Quick Start

### Option 1: Standalone (no backend)
```bash
cd ev-ai/frontend
python3 -m http.server 8080
# Open http://localhost:8080
```

### Option 2: Full Stack (with 3D model backend)
```bash
# Generate 3D model
python3 generate_model.py

# Start backend
backend/venv/bin/python backend/main.py

# Start frontend (separate terminal)
cd frontend && python3 -m http.server 3000

# Open http://localhost:3000
```

### Option 3: One command
```bash
chmod +x run.sh
./run.sh
```

## 🛠 Tech Stack

| Layer | Technology |
|-------|-----------|
| **3D Engine** | p5.js WEBGL |
| **3D Model** | Python custom GLTF/OBJ generator (raw mesh construction) |
| **Backend** | FastAPI (Python) — model serving, simulation API |
| **AI Logic** | Custom agent (sensors, path following, state machine) |
| **UI** | HTML5 + CSS3 (design tokens, SVG, backdrop blur) |
| **Video** | Pillow + ffmpeg (synthetic terminal demo) |
| **Deployment** | GitHub Pages (frontend) + Vercel (optional backend) |

## 🔑 API Endpoints

| Endpoint | Description |
|----------|-------------|
| `GET /` | App info |
| `GET /api/model` | 3D EV model (GLB) |
| `GET /api/model?format=obj` | 3D EV model (OBJ) |
| `GET /api/model/info` | Model metadata |
| `GET /api/status` | Simulation state |
| `POST /api/control` | Control commands (reset, camera) |
| `GET /api/sensors` | Sensor readings |
| `GET /api/health` | Health check |
